### Component Visual Specs

**HistoryButton (header icon)**
- Position: `absolute left-4 top-1/2 -translate-y-1/2` (mirrors ShareButton on the right)
- Size: `min-h-[44px] min-w-[44px] flex items-center justify-center`
- Icon: Inline SVG clock icon, 20x20, `stroke-current` with 2px stroke width
- Color: `text-gray-500 hover:text-gray-600 dark:text-gray-400 dark:hover:text-gray-300`
- Interaction: `hover:bg-gray-100 dark:hover:bg-gray-800 rounded-lg active:scale-95 transition-transform`

**HistoryDrawer**
- Container: `fixed inset-y-0 left-0 z-50 w-80 max-w-[85vw] bg-white dark:bg-gray-900 shadow-xl`
- Animation: `transform transition-transform duration-300 ease-in-out` -- `translate-x-0` when open, `-translate-x-full` when closed
- Backdrop: `fixed inset-0 z-40 bg-black/50` -- click to close
- Header: `p-4 border-b border-gray-200 dark:border-gray-700 flex items-center justify-between`
  - Title: `text-[20px] font-bold leading-[1.2] text-gray-900 dark:text-gray-100` -- text: "History"
  - Close button: `min-h-[44px] min-w-[44px] flex items-center justify-center text-gray-500 dark:text-gray-400` -- X icon (inline SVG or `&times;` entity at 24px)
- Entry list: `flex-1 overflow-y-auto p-4 space-y-2`
- Each entry: `p-4 rounded-lg bg-gray-50 dark:bg-gray-800 border border-gray-200 dark:border-gray-700 min-h-[44px] cursor-pointer hover:bg-gray-100 dark:hover:bg-gray-700 transition-colors`
  - Bill name: `text-base font-bold text-gray-900 dark:text-gray-100 truncate` -- show `billName` or "Unnamed bill"
  - Row below name: `flex items-center justify-between mt-1`
    - Date: `text-sm text-gray-500 dark:text-gray-400` -- formatted as "Mar 19, 4:30 PM"
    - Total: `text-sm text-gray-500 dark:text-gray-400 font-bold` -- formatted as "$XX.XX"

**RestoreConfirmDialog**
- Follows `ShareFallbackModal.tsx` pattern exactly
- Backdrop: `fixed inset-0 bg-black/50 z-[60]` -- click to cancel
- Dialog: `fixed top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2 z-[70] bg-white dark:bg-gray-800 rounded-xl p-6 max-w-sm w-[calc(100%-32px)] shadow-xl`
- Title: `text-[20px] font-bold leading-[1.2] text-gray-900 dark:text-gray-100 mb-4` -- text: "Load this bill?"
- Button row: `flex justify-end gap-4`
  - Cancel: `min-h-[44px] px-4 text-base font-bold text-gray-500 dark:text-gray-400`
  - Load: `min-h-[44px] px-4 text-base font-bold text-blue-500 dark:text-blue-400`
- Keyboard: Escape key dismisses (cancel behavior)

## Interaction Contracts

| Interaction | Behavior |
|-------------|----------|
| Tap history icon | Opens drawer with slide-in animation (300ms ease-in-out). Backdrop appears. |
| Tap backdrop (drawer) | Closes drawer with slide-out animation. |
| Tap close button (drawer) | Closes drawer with slide-out animation. |
| Tap history entry | Opens RestoreConfirmDialog centered over the drawer+backdrop. |
| Tap "Load" in dialog | Close dialog, close drawer, load bill into editor, start new session, show "Bill loaded" toast. |
| Tap "Cancel" in dialog | Dismiss dialog. Drawer stays open. Current bill unchanged. |
| Press Escape (dialog open) | Same as tap "Cancel". |
| Press Escape (drawer open, no dialog) | Close drawer. |

## State Descriptions

| State | What the User Sees |
|-------|--------------------|
| Drawer empty | "No saved bills yet. Bills are auto-saved as you edit." centered in the drawer content area. Gray-500 text, text-base size. |
| Drawer with entries | Scrollable list of up to 10 entries, newest at top. Each shows bill name (or "Unnamed bill"), date, and total. |
| Confirm dialog open | Centered modal with "Load this bill?" title and Load / Cancel buttons. Drawer remains visible but dimmed behind z-60 backdrop. |
| Post-restore | Drawer closes, editor shows the restored bill, "Bill loaded" toast appears at bottom-center for 2 seconds. |

## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| Tap targets | All buttons and history entries: `min-h-[44px]` (project standard) |
| Keyboard dismiss | Escape closes dialog (cancel) and drawer |
| Focus trap | Not required for drawer (simple overlay pattern matches existing ShareFallbackModal) |
| Screen reader | History button: `aria-label="Open history"`. Close button: `aria-label="Close history"`. Each entry: role not needed (native button or clickable div with role="button" and descriptive aria-label). |
| Color contrast | All text meets WCAG AA -- gray-500 on white is 4.6:1, gray-400 on gray-900 is 5.0:1. Blue-500 on white is 4.5:1. |
